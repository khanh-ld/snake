# Snake Web Game

Game Snake chạy bằng HTML/CSS/JavaScript thuần, phù hợp deploy miễn phí với GitHub Pages.

## Chạy local
Mở file `index.html` trực tiếp trên trình duyệt.

## Deploy GitHub Pages
1. Push code lên GitHub.
2. Vào repository `Settings` -> `Pages`.
3. Ở `Build and deployment`, chọn:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` và thư mục `/ (root)`
4. Save và chờ khoảng 1-3 phút.
5. URL sẽ có dạng:
   - `https://<username>.github.io/<repository>/`
